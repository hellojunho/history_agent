| 항목 | 내용 |
|------|------|
| **건국** | 918년 왕건 |
| **도읍** | 개경(開京, 현 개성) |

| 시기 | 왕 | 특징 |
|------|----|------|

## 2. 건국과 후삼국 통일

### 🔴 태조 왕건 (재위 918~943)

> **핵심 전략: 호족 통합(회유) + 견제의 이중 정책**

#### 호족 통합(회유) 정책

| 정책 | 내용 |
|------|------|
| **혼인 정책** | 유력 호족의 딸과 결혼 (29명의 왕비) → 호족과 결탁 |
| **사성 정책(賜姓政策)** | 호족에게 **왕씨 성** 하사 → 왕족으로 편입 |
| **역분전(役分田)** | 후삼국 통일 공신에게 공로에 따라 토지 지급 |

#### 호족 견제 정책

| 정책 | 내용 |
|------|------|
| **사심관 제도(事審官)** | 지방 호족 세력을 회유·감시하기 위해 중앙 고위 관리를 출신 고향의 사심관으로 임명하여 향리 감시와 치안 책임을 지게 함 (최초: 신라 경순왕 김부) |
| **기인 제도(其人制度)** | 지방 호족(향리)의 세력을 통제하기 위해 그 자제를 수도인 개경에 머물게 하여 볼모(인질)로 삼은 제도 (신라의 상수리 제도 계승) |

#### 민생 안정 정책

| 정책 | 내용 |
|------|------|
| **흑창(黑倉)** | 빈민 구제 창고 (의창의 전신) |
| **취민유도(取民有度)** | 세율 1/10로 제한 → 민생 안정 |

#### 북진 정책

| 정책 | 내용 |
|------|------|
| **고구려 계승** | 국호를 "고려(高麗)"로 하여 고구려 계승 표방 |
| **서경 중시** | 평양을 서경으로 중시, 북진의 거점 |
| **발해 유민 수용** | 926년 발해 멸망 후 유민을 적극적으로 받아들임 |

#### 🔴 훈요 10조 (訓要十條, 943년)

> 태조가 후손에게 남긴 10가지 유훈 — 국가 운영 원칙

| 조항 | 핵심 내용 |
|------|-----------|
| 1조 | 불교의 힘으로 나라를 세웠으니 불교를 숭상하라 |
| 2조 | 사원은 도선의 풍수설에 따라 지어라 (임의 증설 금지) |
| 4조 | **중국 제도 무조건 모방 금지** — 거란은 금수의 나라이니 본받지 말라 |
| 5조 | 서경(평양)을 중시하라 |
| 6조 | 연등회·팔관회를 소홀히 하지 말라 |
| 8조 | **차현(車峴) 이남 공주강(금강) 외 사람을 등용하지 말라** |
| 10조 | 경사(유교 경전)를 널리 읽어 옛것을 거울삼아라 |

> **한능검 최빈출 왕!** 노비안검법 + 과거제 세트로 반드시 암기

| 연도 | 정책 | 목적/의의 |

| **958** | **과거제(科擧制) 실시** | 후주 출신 **쌍기(雙冀)** 의 건의 → 유학적 능력 기준 관리 선발, 호족 중심 관료제 개편 |
| **960** | **공복 제정(公服制定)** | 관리 위계 복색 규정 (자·단·비·녹색) → 관료 질서 확립 |
| - | **칭제건원(稱帝建元)** | 황제국 체제 — 연호 **광덕(光德)**, **준풍(峻豊)** 사용 |
| - | **호족 숙청** | 공신 세력·호족 대거 숙청 → 강력한 왕권 확립 |
| - | 제위보 설치 | 빈민 구제 기금 |

### 🔴 인종 (재위 1122~1146) — 문벌 귀족의 모순 폭발

| 연도 | 사건 | 내용 |
|------|------|------|
| **1126** | **이자겸의 난(李資謙의 亂)** | 왕의 외조부 이자겸이 반란 → 인종이 진압, 이자겸 유배 |
| **1135** | **묘청의 서경 천도 운동** | 승려 묘청 등이 서경(평양) 천도 주장, '칭제건원'·금나라 정벌 주장 → 인종이 거부 → 묘청, 서경에서 반란(대위국 건국 선포) → **김부식이 관군 이끌고 진압** |
| **1145** | **삼국사기(三國史記)** 편찬 | 김부식 주도, 현존 최고(最古) 관찬 정사, 기전체 |

> **이자겸의 난과 묘청의 난의 의의:**
> - 두 사건 모두 문벌 귀족 사회의 모순을 드러낸 사건
> - 묘청의 서경 천도 운동: **자주파(묘청·서경 세력)** vs **사대파(김부식·개경 문벌)** 의 대립
> - 신채호는 묘청의 난을 "조선 역사 일천년래 제일대사건"이라 평가

### 🔴 삼별초 항쟁 (1270~1273)

> **배경:** 1270년 개경 환도(무신 정권 붕괴) → 삼별초가 이에 반발

| 단계 | 내용 |
|------|------|
| **1270** | 배중손이 삼별초 이끌고 **진도**로 이동, 항쟁 계속 |
| **1271** | 진도 함락 → 김통정이 **제주도(탐라)**로 이동 |
| **1273** | 여몽 연합군에 의해 제주도 삼별초 완전 진압 |

> **의의:** 대몽 항쟁 정신 + 고려 자주성 표현 / **한계:** 민중의 지지 미흡

#### 🔴 팔만대장경 (八萬大藏經, 1236~1251)

## 7. 고려 통치 체제

### 🔴 중앙 정치 기구

```
2성(二省)
 ├── 중서문하성(中書門下省) — 최고 정무기관, 문하시중이 수장
 │    ├── 재신(宰臣, 2품 이상) — 국가 중요 사안 결정
 │    └── 낭사(郎舍, 3품 이하) — 간쟁(諫爭)·봉박(封駁) 기능
 └── 상서성(尙書省) — 실무 집행
      └── 6부(六部): 이·병·호·형·예·공부

독자 기구
 ├── 중추원(中樞院) — 군사 기밀(추밀) + 왕명 출납(승선)
  ├── 삼사(三司) — 화폐·곡식의 출납·회계 담당
  │    └── **[중요 차이점]** 조선의 삼사(사헌부·사간원·홍문관)는 언론·비판 및 학술 기구이나, 고려의 삼사는 권력과 무관하게 단순 재정 실무 회계만을 담당한 금융 기구입니다.
 ├── 어사대(御史臺) — 관리 비리 감찰
 ├── 도병마사(都兵馬使) — 재신+추밀의 합좌 — 국방·군사 논의 → 후기 도평의사사로 확대
 └── 식목도감(式目都監) — 재신+추밀의 합좌 — 법제·격식 제정
```

> **도병마사·식목도감의 중요성:**
> - 재신(중서문하성)+추밀(중추원)의 **합좌 기구(재추 회의)** → 고려 독자적 제도
> - 당·송에는 없는 고려만의 독자적 정치 기구

## 11. ⭐ 빈출 개념

### ⭐ 광종 세트 (최빈출)
```
노비안검법(956) → 과거제 실시(958, 쌍기 건의) → 공복 제정(960) → 칭제건원(광덕·준풍)
```

### ⭐ 성종 세트
```
최승로 시무 28조(982) → 12목 설치(983) → 국자감 정비(992) → 서희 외교담판(993)
```

### ⭐ 무신 정권 기구 비교
| 기구 | 인물 | 기능 |
|------|------|------|
| 교정도감 | 최충헌 | 최고 권력 기구 |
| 정방 | 최우 | 인사 행정 |
| 도방 | 최충헌 | 사병 조직 |
| 삼별초 | 최우 | 특수 부대 |

### ⭐ 전시과 3단계
```
시정 전시과(976, 경종) → 개정 전시과(998, 목종) → 경정 전시과(1076, 문종)
```

### ⭐ 역사서 비교
```
삼국사기(1145, 김부식, 기전체, 관찬) vs 삼국유사(1281, 일연, 기사본말체, 사찬)
```

### ⭐ 의천 vs 지눌
```
의천 → 교관겸수 → 천태종 (교종 중심)
지눌 → 정혜쌍수·돈오점수 → 조계종 (선종 중심)
```

### ⭐ 대장경 비교
```
초조대장경(현종) → 팔만대장경(고종, 1236~1251) → 직지심체요절(1377, 최고 금속활자 인쇄본)
```

## 12. 🔴 핵심 개념

### 🔴 고려의 독자성

| 항목 | 고려의 독자적 특징 |
|------|-------------------|
| 정치 기구 | **도병마사·식목도감** (당·송에 없는 고려 독자 기구) |
| 외교 | **칭제건원** (황제국 체제 유지) — 광종의 연호 사용 |
| 지방 | **향·부곡·소** (고려 특수 행정 구역) |
| 군사 | **삼별초** (고려 독자 군사 조직) |

### 🔴 문벌 귀족 사회의 특징

- **음서제:** 5품 이상 관리 자손에게 시험 없이 관직 부여
- **공음전:** 세습 가능한 토지 (전시과와 달리 반납 불필요)
- **통혼권:** 유력 가문끼리만 혼인 → 폐쇄적 지배층 형성
- **대표 가문:** 경원 이씨(이자겸), 해주 최씨(최충), 경주 김씨

### 🔴 신진사대부의 성장

- **배경:** 공민왕의 성균관 개편, 과거제 강화
- **특징:** 지방 향리 출신, 과거를 통한 관직 진출, 성리학 수용
- **역할:** 고려 말 위화도 회군 후 이성계와 함께 조선 건국 주도
- **대표 인물:** 정도전, 조준, 정몽주(온건파) 등
