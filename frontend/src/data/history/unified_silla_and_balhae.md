## 1. 시대 개관

### 남북국 시대란?

**남북국 시대(南北國時代)** 란 676년 신라의 삼국통일 이후부터 935년(신라 멸망)·926년(발해 멸망) 전후까지, 한반도 남부의 **통일신라**와 만주·한반도 북부의 **발해**가 공존했던 시기를 이르는 개념이다.

> 조선 후기 실학자 **유득공(柳得恭)** 이 『발해고(渤海考, 1784)』에서 최초로 **"남북국"** 이라는 용어를 사용하며 발해를 우리 역사로 편입할 것을 주장하였다.

| 구분 | 통일신라           | 발해               |
| ---- | ------------------ | ------------------ |
| 위치 | 한반도 남부        | 만주 + 한반도 북부 |
| 건국 | 676년(실질적 통일) | 698년(대조영)      |
| 멸망 | 935년(경순왕)      | 926년(거란에 멸망) |
| 계승 | 신라               | 고구려             |

### 2-3. 경제

#### 🔴 민정 문서 (촌락 문서)

- 1933년 일본 도다이지(東大寺) 쇼소인(正倉院)에서 발견
- **서원경(청주) 인근 4개 촌락**의 실태 기록
- 기록 내용: 호구(인구), 토지 면적, 소·말 수, 뽕나무·잣나무 수까지 상세 기록
- 작성 주기: **3년마다** 촌주(村主)가 작성
- **목적:** 조세·공물·역역 수취의 기준 자료

#### 상업

| 시장           | 위치                                |
| -------------- | ----------------------------------- |
| **동시(東市)** | 경주 동쪽 (지증왕 490년 설치, 최초) |
| **서시(西市)** | 경주 서쪽 (효소왕 때 설치)          |
| **남시(南市)** | 경주 남쪽 (후기 설치)               |

> **시전(市典):** 시장 관리 감독 기구
> **대외 무역:** 당·일본 등과 교역, 울산항(국제 무역항), 장보고의 청해진(828) 설치

## 3. 발해

### 3-1. 건국과 왕별 업적

#### 🔴 건국 — 대조영 (698년)

> **원인:** 고구려 멸망(668) → 고구려 유민과 말갈족이 당의 지배에 저항
> **전개:** 696년 거란족 이진충의 난 → 혼란을 이용해 **대조영**이 동모산(東牟山, 현 만주 길림성 돈화시)에서 건국
> **국호:** 처음 "진(震)" → 이후 "발해(渤海)"
> **성격:** 지배층 고구려인 + 피지배층 말갈인의 다민족 국가

| 고구려 계승 근거 | 내용                       |
| ---------------- | -------------------------- |
| 일본 외교 문서   | 스스로 "고려국왕"이라 칭함 |
| 지배층 구성      | 고구려 유민 중심           |
| 문화             | 온돌 문화, 이불병좌상 양식 |

#### 발해 멸망 (926년)

> **거란(耶律阿保機)** 의 침입 → 상경 함락 → 발해 멸망
> → 발해 유민 일부가 **고려**로 귀순 (왕건이 적극 수용)

---

### 3-2. 통치 제도

#### 🔴 중앙 — 3성 6부 (당제 수용, 독자 운영)

```
3성(三省)
 ├── 정당성(政堂省) → 最高 정무기관 (당의 상서성에 해당) — 대내상이 수장
 ├── 선조성(宣詔省) → 왕명 반포 (당의 중서성)
 └── 중대성(中臺省) → 정책 심의 (당의 문하성)

6부(六部) — 정당성 산하
 ├── 충부(忠部) — 이부에 해당
 ├── 인부(仁部) — 호부에 해당
 ├── 의부(義部) — 예부에 해당
 ├── 지부(智部) — 병부에 해당
 ├── 예부(禮部) — 형부에 해당
 └── 신부(信部) — 공부에 해당
```

> 당의 명칭(이·호·예·병·형·공부)을 **유교 덕목(충·인·의·지·예·신)** 으로 바꾸어 운영 → 독자성 표현

#### 지방 — 5경 15부 62주

| 구분     | 내용                                                             |
| -------- | ---------------------------------------------------------------- |
| **5경**  | 상경(용천부)·중경(현덕부)·동경(용원부)·남경(남해부)·서경(압록부) |
| **15부** | 각 지역 광역 행정 단위                                           |
| **62주** | 하위 행정 단위                                                   |
| 촌락     | 말갈 촌락 자치 허용                                              |

#### 군사

- **10위(衛):** 중앙군 편성
- 지방은 부·주별 군사 조직 운용

## 6. 후삼국 성립

### 배경

> 신라 하대(下代, 780년 이후): 왕위 쟁탈전 → 지방 통제 약화 → 호족 세력 대두 → 농민 봉기

| 연도 | 사건                                            |
| ---- | ----------------------------------------------- |
| 889  | 원종·애노의 난 (사벌주) — 최초 전국적 농민 봉기 |
| 891  | 양길의 봉기 (북원경 원주)                       |
| 900  | 견훤, 후백제 건국                               |
| 901  | 궁예, 후고구려 건국                             |
| 918  | 왕건, 고려 건국                                 |
| 926  | 발해 멸망 (거란)                                |
| 935  | 신라 경순왕 고려에 귀순                         |
| 936  | 왕건, 후삼국 통일                               |

### 🔴 견훤 (甄萱) — 후백제 (900~936)

- **출신:** 신라 아잔 출신, 전라도 지역 장악
- **건국:** 900년, **완산주(전주)** 에서 후백제 건국
- **특징:** 백제 계승 의식, 신라에 강경 대응
- **주요 사건:**
  - 927년 신라 경애왕 살해, 금성(경주) 약탈
  - 930년 고창 전투에서 왕건에 패배
  - 935년 아들 **신검**에게 왕위 찬탈당함 → 왕건에 귀순
  - 936년 **일리천 전투** 에서 왕건에 의해 후백제 멸망

### 🔴 궁예 (弓裔) — 후고구려 → 마진 → 태봉 (901~918)

- **출신:** 신라 왕족 출신(헌안왕·경문왕의 서자로 전해짐)
- **변천:**
  - 901년 **후고구려** 건국 (송악=개성)
  - 904년 **마진(摩震)** 으로 국호 변경, 철원 천도
  - 911년 **태봉(泰封)** 으로 개칭
- **특징:** 미륵 신앙 이용한 전제 통치, 말년에 폭정
- **멸망:** 918년 부하 장수들에 의해 폐위 → **왕건** 이 추대되어 고려 건국

### 🔴 왕건 (王建) — 고려 건국 (918)

- 918년 고려 건국, 연호 천수(天授), 송악(개성)에 도읍
- 935년 신라 경순왕 귀순 → 경순왕을 경주 사심관으로 임명
- 936년 후백제 멸망 → **후삼국 통일 완성**

## 7. ⭐ 빈출 개념

### ⭐ 신문왕 세트 (한능검 최빈출)

```
김흠돌의 난 진압(681) → 국학 설치(682) → 9주5소경(685)
→ 관료전 지급(687) → 녹읍 폐지(689) → 9서당·10정 편성
```

### ⭐ 녹읍의 역사

```
녹읍 폐지(신문왕, 689) → 녹읍 부활(경덕왕, 757)
```

### ⭐ 독서삼품과 (788, 원성왕)

- 목적: 유교 경전 능력으로 관리 선발
- 한계: 골품제 장벽으로 실효성 미흡

### ⭐ 민정 문서 (촌락 문서)

- 서원경 4개 촌락, 3년마다 작성, 일본 쇼소인에서 발견

### ⭐ 발해 왕 - 연호 매칭

| 왕   | 연호       |
| ---- | ---------- |
| 무왕 | 인안(仁安) |
| 문왕 | 대흥(大興) |
| 선왕 | 건흥(建興) |

### ⭐ 발해의 고구려 계승 근거

```
① 대조영 = 고구려 유민
② 일본에 보낸 외교 문서 — 스스로 "고려국왕" 칭함
③ 온돌 문화
④ 이불병좌상 (고구려 양식)
⑤ 정혜 공주 묘 (고구려식 굴식 돌방무덤)
```

### ⭐ 무구정광대다라니경

- **세계 최고(最古) 목판 인쇄물**
- 불국사 석가탑(삼층석탑)에서 발견(1966)
- 751년경 제작 추정

## 8. 🔴 핵심 개념

### 🔴 통일신라 시대 구분

| 구분           | 시기                    | 특징                         |
| -------------- | ----------------------- | ---------------------------- |
| **상대(上代)** | ~무열왕 이전            | 성골 왕, 화백회의 강력       |
| **중대(中代)** | 무열왕~혜공왕 (654~780) | 진골 왕, 전제 왕권 전성기    |
| **하대(下代)** | 선덕왕~경순왕 (780~935) | 왕위 쟁탈전, 호족 대두, 쇠퇴 |

### 🔴 골품제의 모순

- 6두품의 관직 상한선(아찬, 6관등) → 능력 있어도 승진 불가
- 신라 말 6두품 지식인들이 호족과 결탁 → 고려 건국 세력으로 참여

### 🔴 발해 3성 6부의 독자성

- 당제를 받아들이되, 6부 명칭을 유교 덕목(충·인·의·지·예·신)으로 변경
- 최고 기관을 **정당성**으로 설정 (당의 상서성에 해당)
